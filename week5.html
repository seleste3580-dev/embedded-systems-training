Week 5 - Advanced Pico Programming, Sensors, and LCD Displays

Objectives

The main objectives of the fifth week of the embedded systems training program were to:

  • Continue exploring and programming the Raspberry Pi Pico using intermediate techniques.
  • Implement logic assignments, focusing on crucial embedded concepts such as switch debouncing algorithms.
  • Further integrate peripheral devices, specifically ultrasonic sensors and LCD displays.
  • Deepen familiarity with CircuitPython and the Thonny IDE for rapid prototyping.

Overview

Key learning outcomes from Week 5:

  • Developed and implemented software debouncing algorithms to ensure reliable input readings from mechanical switches.
  • Successfully interfaced an LCD display with the Raspberry Pi Pico to create visual readouts for sensor data.
  • Combined ultrasonic sensors with the LCD display to create a real-time distance matching tool.
    • Gained greater proficiency in utilizing CircuitPython and Thonny IDE to write, debug, and deploy embedded code effectively.
