Week 10 - RFID Access Control Systems with ESP32

Objectives

The main objectives of the tenth week of the embedded systems training program were to:

  • Transition RFID concepts learned previously to the ESP32 platform.
  • Interface the MFRC522 RFID module with the ESP32 using the Arduino IDE.
  • Design and implement a functional Access Control System project.
  • Combine hardware integration with conditional logic to grant or deny access based on scanned RFID tags.

Overview

Key learning outcomes and activities from Week 10:

  • Successfully wired the RFID reader to the ESP32 using the SPI interface.
  • Developed a comprehensive program that authenticates scanned RFID cards against a database of authorized UIDs.
  • Built an access control project that triggers outputs (such as LEDs, buzzers, or relays) upon successful tag verification.
  • Explored real-world applications of these systems, reinforcing our understanding of secure entry mechanisms in IoT devices.
