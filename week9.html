Week 9 - RFID Integration with Raspberry Pi Pico W

Objectives

The main objectives of the ninth week of the embedded systems training program were to:

  • Understand the fundamentals of Radio Frequency Identification (RFID) technology.
  • Interface an RFID module (e.g., MFRC522) with the Raspberry Pi Pico W.
  • Write code to read from and write to RFID tags and cards.
  • Explore communication protocols (like SPI) used between the microcontroller and the RFID module.

Overview

Key learning outcomes and activities from Week 9:

  • Gained hands-on experience wiring the RFID reader module to the specific SPI pins of the Raspberry Pi Pico W.
  • Learned how to import and utilize specific libraries for handling RFID communication.
  • Successfully read Unique Identifiers (UIDs) from various RFID tags and printed them to the serial monitor.
  • Discussed the security aspects and practical use cases of RFID systems in modern embedded applications.
