# Embedded Systems Training Documentation

This repository serves as a comprehensive documentation of my weekly progress, experiments, hardware development, and system design during the Embedded Systems Training program under the **AFRETEC** program at **Gearbox-Europlacer**.

The goal of this training is to gain deep knowledge and practical skills in embedded systems, ranging from low-level C programming to advanced IoT development with ESP32.

## 🚀 Project Overview

The project is structured as a weekly log, each documented in a dedicated HTML report. It covers:
- Microcontroller architecture and programming.
- Sensor integration and data acquisition.
- Communication protocols (UART, I2C, SPI).
- Real-world embedded applications and algorithms.

## 🛠️ Technology Stack & Hardware

- **Microcontrollers:** Raspberry Pi Pico (RP2040), ESP32.
- **Languages:** C, CircuitPython.
- **Tools:** Thonny IDE, Arduino IDE, ESP-IDF (Espressif IoT Development Framework).
- **Peripherals:** Ultrasonic Sensors (HC-SR04), LCD Displays, Switches, and LEDs.
- **Concepts:** Interrupts (ISR), Debouncing Algorithms, Digital/Analog Communication.

## 📅 Weekly Progress Summary

- **Week 1:** Introduction to Embedded Systems fundamentals.
- **Week 2:** Deep dive into C Programming for embedded applications.
- **Week 3:** Microcontroller programming basics and practical simulations.
- **Week 4:** Raspberry Pi Pico programming, Ultrasonic sensor integration, Interrupts, and Protocols (UART, I2C, SPI).
- **Week 5:** Advanced Pico programming with CircuitPython, LCD interfacing, and implementation of Debouncing Algorithms.
- **Week 6:** Transition to ESP32 platform, setup of Arduino IDE & ESP-IDF, and in-depth datasheet analysis.

## 🖥️ How to View

2. Open `index.html` in any modern web browser.
